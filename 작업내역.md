## 지원 기능

### 1. URL 분석
- 텔레그램에 URL 전송 → 웹페이지 스크래핑 → Claude 분석 → 옵시디언 노트 생성
- 스크래핑 성공 기준: 에러 없음 + 본문 100자 초과
- SNS 링크 폴백: 스크래핑 실패/부족 시 Claude가 WebFetch/WebSearch로 직접 URL 읽어서 분석
- YouTube, Instagram, Threads, Facebook 등 SNS 링크 처리 가능
- 분석 실패 시 옵시디언에 저장하지 않고, 실패 URL을 마지막에 일괄 보고

### 2. 텍스트 분석
- 텍스트 전송 → Claude가 내용 정리 → 옵시디언 노트 생성
- 분석 실패 시 저장 스킵 + 실패 보고

### 3. 이미지 분석
- 이미지 전송 → Claude Code가 Read 도구로 이미지 읽기/분석 → 옵시디언 노트 생성
- 가장 큰 해상도의 사진 자동 선택
- 원본 이미지를 vault의 `attachments/` 폴더에 복사 (`img_YYYYMMDD_HHMMSS.jpg`)
- 노트에 `![[이미지파일명]]`으로 임베드
- 이미지 내 텍스트 추출 포함

### 4. txt 파일 일괄 처리
- `.txt` 파일 업로드 → 파싱 → 큐 기반 병렬 처리 (동시 3개)
- 인코딩 자동 감지: UTF-8 → CP949 → EUC-KR → Latin-1 순서로 시도
- **자동 구분 규칙 감지:**
  - 2줄 이상 빈 줄이 존재하면 → 2줄 빈 줄로 게시글 구분 (1줄 빈 줄은 같은 게시글 내 줄바꿈)
  - 2줄 빈 줄이 없으면 → 1줄 빈 줄로 게시글 구분
- URL과 텍스트 혼합 가능

### 5. 카카오톡 대화 내보내기 자동 감지/파싱
- txt 파일 업로드 시 카카오톡 형식 자동 감지 (날짜 구분선, `[이름] [시간]` 패턴)
- 날짜 구분선 및 메시지 메타데이터(`[이름] [오전/오후 시:분]`) 자동 제거
- 사진/동영상/파일 등 텍스트 처리 불가 항목 스킵
- 관련 메시지 자동 그룹핑: URL + 짧은 설명을 하나의 게시글로 묶음

### 6. 노트 평가 시스템
- 노트 저장 후 자동 평가 (Claude Code CLI 호출)
- 평가 항목 6개: 최신성(1-5), 실용성(1-5), 신뢰도(1-5), 깊이(1-5), 개발자 적합성(1-5), Claude Code 적용성(1-5)
- 종합등급 (30점 만점): A(25+), B(19-24), C(13-18), D(12이하)
- 평가 결과를 옵시디언 노트 하단에 자동 추가 (별점 + 한줄평)
- 원본 콘텐츠 저장: 분석 요약본과 함께 원본도 옵시디언 접이식 callout(`[!quote]- 원본 보기`) 형식으로 저장
- Claude Code 적용 가능한 팁 발견 시 텔레그램에서 인라인 버튼(Global 반영/Skill 제작/팁 저장/스킵) 4분기 승인 요청 (2x2 레이아웃) → Global 반영 시 `~/.claude/CLAUDE.md`에 추가, Skill 제작 시 `~/.claude/commands/`에 슬래시 커맨드 .md 파일 자동 생성, 팁 저장 시 `~/.claude/tips/`에 개별 .md 파일로 풀에 저장 (frontmatter에 source/date/applied 메타데이터)
- `/apply-tips` 슬래시 커맨드: 팁 풀(~/.claude/tips/)에서 미적용 팁 수집 → 현재 프로젝트 기술스택 분석 → 적합 팁 매칭(적합도 1-5) → 사용자 확인 후 프로젝트 CLAUDE.md에 적용
- 텔레그램에 등급과 Claude Code 적용 여부 알림

### 7. 내용 수준 중복 비교
- 새 콘텐츠 저장 전 기존 노트와 내용 비교 (Claude Code CLI 호출)
- 판정: 신규(새 노트 생성) / 중복(저장 스킵) / 보강(기존 노트에 새 정보만 추가)
- URL이 다르더라도 내용이 같으면 중복으로 판정
- URL 분석, 텍스트 분석 모두 적용
