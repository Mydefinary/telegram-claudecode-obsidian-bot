========================================
 Changelog (2026-04-06)
========================================

[Part 1: Error Log Analysis & Stability]

1. [Global Error Handler] Registered error_handler in bot.py
   - Handles Conflict (bot instance collision), NetworkError, TimedOut, RetryAfter
   - Conflict logged as ERROR (message loss risk), others as WARNING
   - Added drop_pending_updates=True to prevent restart conflicts

2. [Claude CLI Timeout Retry] analyzer.py _run_claude_cli
   - 1 retry on timeout: 180s first attempt → 300s retry
   - Process killed on timeout to prevent zombie processes

3. [YouTube Gemini Fallback] analyzer.py analyze_youtube
   - Falls back to Claude CLI analyze_link_direct when Gemini fails
   - Handles google-genai not installed / token limit exceeded

4. [Scraper 403 Retry] scraper.py fetch_page_content
   - Retries with alternate User-Agent on 403 Forbidden
   - Added Accept/Accept-Language headers

[Part 2: Security Patch]

5. [User Whitelist] ALLOWED_USER_IDS environment variable
   - config.py: Parses comma-separated Telegram user IDs
   - bot.py: _is_user_allowed() check on all handlers
   - Empty = allow all (backwards compatible)

6. [Prompt Injection Defense] XML tag isolation
   - analyzer.py: All user inputs wrapped in <user_input> tags
   - Separates system prompt from user-provided content

7. [Error Message Sanitization] Internal info hidden from users
   - bot.py: Replaced {e} in user-facing error messages with generic text
   - Detailed errors logged only to log files

8. [SSRF Protection] Internal IP blocking
   - scraper.py: _is_safe_url() validates URL targets
   - Blocks private, loopback, link-local, multicast IPs via DNS resolution

9. [File Size Limit] DoS prevention
   - config.py: MAX_FILE_SIZE (default 10MB)
   - bot.py: Rejects oversized file uploads before download

10. [Log URL Masking] Query string sanitization
    - bot.py: _mask_url() strips query params from logged URLs
    - Prevents API keys in URLs from leaking to log files

11. [Terminal Popup Fix] Windows console window suppression
    - analyzer.py: CREATE_NO_WINDOW flag for Claude CLI subprocess
    - Prevents console popups during background PM2 operation
